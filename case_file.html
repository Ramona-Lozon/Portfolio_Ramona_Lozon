Earbuds

this is a project i did while enrolled at fanshawe in my interactive media design program, where we set out to design a pair of earbuds with a case

for this project, we needed to take our designs, 3-d model them, and build a website with an interactive interface where users could see

we started in the conceptual stage, designing the look of the earbuds and the case. here are some concepts that i took into Cinema 4d

after modeling the earbuds in C4d, we made promotional images, set up hot spots and prepared to create an x-ray view of the internal components

the final product is a website that was built using SASS, Javascript, and Greensock, for functionality

by the end of the project we had a working website that acted as an advertising page for the earbuds, displaying their features using a 3d model viewer and showing their internal workings using a slider created with SASS and Greensock.
